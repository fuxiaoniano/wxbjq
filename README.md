# 傅小念的编辑器

一个用于微信公众号图文排版的本地富文本编辑器。项目提供可视化正文编辑、HTML 源码编辑、系统模板文件保存、草稿列表管理、链接插入和一键复制能力，适合在本地完成文章排版后粘贴到微信公众号后台。

## 功能

- 可视化编辑微信公众号正文内容
- 界面采用纯色背景、圆角描边按钮、薄荷绿主界面和固定红色模板中心
- 支持加粗、斜体、下划线、列表、对齐和文字颜色，可直接输入六位颜色编码
- 支持左侧模板样式卡片预览，点击后自动添加到正文
- 内置红色系通用系统模板
- 支持导入 HTML 模板，并通过本地服务写入网站级系统模板文件
- 支持 HTML 源码模式编辑
- 支持插入链接
- 支持保存多个草稿，在弹窗列表中查看、恢复和删除草稿文件
- 支持复制为富文本内容，便于粘贴到公众号后台

## 项目结构

```text
.
├── app.js       # 编辑器交互逻辑
├── index.html   # 页面结构
├── server.js    # 本地静态文件服务器和数据接口
├── system-templates.json # 导入后写入的网站级系统模板
├── styles.css   # 页面样式
├── README.md    # 项目说明
└── data/         # 运行后自动创建，保存草稿文件
```

## 本地运行

需要先安装 Node.js。

```powershell
cd "C:\Users\Administrator\Documents\微信编辑器"
node server.js
```

启动后在浏览器打开：

```text
http://127.0.0.1:8090
```

如果需要换端口：

```powershell
$env:PORT=8080
node server.js
```

## 使用方式

1. 在编辑区输入正文内容。
2. 使用顶部工具栏调整文字样式。
3. 在左侧模板中心点击模板样式卡片，模板会自动添加到正文内容中。
4. 粘贴 HTML 片段并保存后，模板会写入 `system-templates.json`，可随网站文件一起迁移到其他设备。
   双击打开 `index.html` 使用时，浏览器会要求选择一次项目里的 `system-templates.json` 才能写入。
5. 需要直接编辑 HTML 时，点击编辑源码。
6. 点击保存草稿可把当前内容保存为一个新草稿。
7. 点击草稿列表可在弹窗中查看、恢复或删除已保存草稿。
8. 点击复制正文后，粘贴到微信公众号后台编辑器。

## 更新约定

后续每次修改项目代码时，需要同步完成：

1. 更新相关代码文件。
2. 更新 README.md 中受影响的说明。
3. 提交本地 Git 记录。
4. 上传到 GitHub 仓库。
5. 验证 GitHub 远端文件已更新。

当前本机普通 `git push` 连接 GitHub 443 端口不稳定，因此已采用 GitHub CLI API 通道同步文件到仓库。后续如果普通 `git push` 恢复可用，可以改回标准 Git 推送流程。
