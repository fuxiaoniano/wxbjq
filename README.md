# 微信编辑器

一个用于微信公众号图文排版的本地富文本编辑器。项目提供可视化正文编辑、HTML 源码编辑、草稿保存、链接插入和一键复制能力，适合在本地完成文章排版后粘贴到微信公众号后台。

## 功能

- 可视化编辑微信公众号正文内容
- 支持加粗、斜体、下划线、列表、对齐和文字颜色
- 支持文章标题和作者信息
- 支持 HTML 源码模式编辑
- 支持插入链接
- 支持本地保存和恢复草稿
- 支持复制为富文本内容，便于粘贴到公众号后台

## 项目结构

```text
.
├── app.js       # 编辑器交互逻辑
├── index.html   # 页面结构
├── server.js    # 本地静态文件服务器
├── styles.css   # 页面样式
└── README.md    # 项目说明
```

## 本地运行

需要先安装 Node.js。

```powershell
cd "C:\Users\Administrator\Documents\微信编辑器"
node server.js
```

启动后在浏览器打开：

```text
http://127.0.0.1:8090
```

如果需要换端口：

```powershell
$env:PORT=8080
node server.js
```

## 使用方式

1. 在编辑区输入正文内容。
2. 使用顶部工具栏调整文字样式。
3. 需要直接编辑 HTML 时，点击 HTML 源码编辑。
4. 点击保存草稿可把当前内容保存到浏览器本地。
5. 点击微信一键复制后，粘贴到微信公众号后台编辑器。
